🌙 Night Moves

Mobile Monero Mining — Earn XMR While You Sleep

⚡ Updated Signup Script — Now with SHA256 hash verification for security!
Step 1

Install Termux

Download Termux from F-Droid (recommended) or Google Play Store.

Scan to download Termux

Get Termux
Step 2

Install Python

Open Termux and run:

pkg update && pkg upgrade
pkg install python
Step 3

Run Signup Script

One command to download, verify, and run:

curl -o signup.py -L https://raw.githubusercontent.com/xmrtdao/mmlauncher/main/scripts/mobile-signup.py && sha256sum signup.py && python3 signup.py
Expected SHA256: 51099f420e25e74997c1de59c2ddaf0f3c40fcebfb3c5b1a07e1e125f5b0d7d0
The sha256sum command will print the hash. Compare it to verify the script is authentic before it runs!
Step 4

Start Mining

The signup script will configure your miner and connect you to the pool. You'll see:

  • Your wallet address
  • Mining hashrate
  • Accepted shares
  • Pending payout
✅ No root required! Termux runs in user space.
