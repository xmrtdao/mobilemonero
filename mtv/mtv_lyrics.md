## Technical Notes

### MiniMax Music API Schema (from mmx-cli v1.0.15 source)

**Valid models**: `music-2.6` (default), `music-2.5+`, `music-2.5`  
**Endpoint**: `POST https://api.minimaxi.chat/v1/music_generation`  
**Auth**: `Authorization: Bearer <key>`  
**JSON payload**:

```json
{
  "model": "music-2.6",
  "prompt": "Music style description (max 2000 chars if combined with flags)",
  "lyrics": "Song lyrics with structure tags. Max 3500 chars. Supported tags: [Intro], [Verse], [Pre Chorus], [Chorus], [Interlude], [Bridge], [Outro], [Post Chorus], [Transition], [Break], [Hook], [Build Up], [Inst], [Solo]",
  "is_instrumental": false,
  "lyrics_optimizer": false,
  "audio_setting": {
    "format": "mp3",
    "sample_rate": 44100,
    "bitrate": 256000
  },
  "output_format": "url",
  "stream": false
}
```

**Response** (success):
```json
{
  "data": {
    "audio_url": "https://..."
  }
}
```

**Errors seen**:
- `1008 insufficient balance` → Token Plan inactive. Activate at platform.minimaxi.com
- `2062 no active token plan subscription` → Same root cause
- `2013 invalid params` → Wrong field or missing required fields (fixed above)

### Token Plan Requirements
- Music-2.6 / Music-2.5+: Requires **Max plan or above**
- See: `https://platform.minimaxi.com/subscribe/token-plan`
